PlayerPrefs 장점

1).유니티에 내장 쉽게 저장 불러오기됨
2).직관적인 함수이다 PlayerPrefs.SetInt("저장",1)
3)Dictionery와 비슷 (C++ 에서는 Map)

PlayerPrefs 단점

1)저장하고 불러올수 있는 자료형이 제한적
(int,float,string만 가능)

2)각각 변수들을 함수로 써서 다 저장과 로드를 해야함
(나중에 100개 넘는 변수들을 저장과 불러오기를 해야되는..)

3)보안도 취약

JSON 장점
1)저장시키고 불러올수 있는 자료형이 다양
2)능력만 되면 암호화가능해서 보안에 도 좋음
JSON 단점

1)저장과 불러오기를 직접코드로 구현(이게단점이야..?)
